Download Page

These download files are generated automaticaly twice a day (5 a.m. and 5 p.m. CEST) , so please check the latest build and the ChangeLog before reporting a bug.
The packages contain both, the JZuul Game Development Kit and the JZuul Engine. For more information, refer to the Docs section on the main page.

To run JZuul you need a Java Runtime Environment (JRE), at least version 1.4.2. You can get it from SUN at their download page.

JZuul is free software as it distributed under the terms of the GNU General Public License Version 2. A German translation of the GPL is also available.

Source

The source files contain libraries for all above platforms, therefore they are a lot bigger. To build JZuul you need a Java Software Development Kit (J2SDK) at least version 1.4.2. You can get it from Sun on their download page.  The source packages come with an ANT build script, you need Apache ANT to run them. You can get it from the Apache ANT download page. Furthermore, the developers recommend Eclipse for Java development.
